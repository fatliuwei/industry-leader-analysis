# 更新日志

所有重要的更改都将记录在此文件中。

## [1.0.0] - 2026-04-15

### 新增
- ✨ 初始版本发布
- ✨ 支持申万2021行业分类数据获取（511个行业）
- ✨ 三层数据补充机制（成分股数据 → 关键词匹配 → 智能搜索）
- ✨ 自动识别行业龙头股（基于市值、ROE等指标）
- ✨ 支持定时任务配置（每日/每周/每月自动更新）
- ✨ 多格式输出（CSV、Excel、Markdown）
- ✨ 命令行工具支持
- ✨ Python API接口
- ✨ 完整文档和示例

### 数据覆盖
- L1级行业：100% (31/31)
- L2级行业：97% (130/134)
- L3级行业：69% (239/346)
- 总体覆盖率：78.3% (400/511)

### 功能特性
- 行业数据获取：支持申万2021分类体系
- 龙头股识别：市值、ROE、净利率等多维度分析
- 财务数据分析：支持ROE、净利率、营收增长等指标
- 定时任务：支持每日、每周、每月自动更新
- 报告生成：自动生成多格式分析报告
- 命令行工具：支持fetch、analyze、report等命令

### 支持环境
- openclaw：支持作为模块导入使用
- opencode：支持命令行和Notebook使用
- claude code：支持定时任务自动执行

### 文档
- README.md：完整使用指南
- QUICKSTART.md：快速开始指南
- INTEGRATION_GUIDE.md：集成指南
- skill.md：Skill主文档
- examples/：使用示例代码
